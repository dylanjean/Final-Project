Welcome to Our Project

Our project uses machine learning to predict Bitcoin worth. Additionally, if you are interested, we have pages for news, a bit of bitcoin history, comparison against gold's worth, and an about our author's page.

About Our Process

With this web application we have created a powerful tool that allows someone looking to invest in Bitcoin to minimize there risk on investment. We have done this by creating a machine learning model that alllows us to predict the the end price of Bitcoin.
